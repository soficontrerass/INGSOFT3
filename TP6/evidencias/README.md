# TP6 — Evidencias

Este documento contiene instrucciones y detalles sobre las evidencias recopiladas durante el desarrollo del proyecto TP6. Las evidencias incluyen capturas de pantalla, logs y resultados de pruebas que demuestran el correcto funcionamiento de la aplicación y su infraestructura.

## Contenido de las evidencias

1. **Capturas de pantalla**: Se incluirán imágenes que muestren el estado de los despliegues, la interfaz de usuario y cualquier otro aspecto relevante del sistema.
2. **Logs de ejecución**: Se documentarán los logs generados durante las pruebas y despliegues, que servirán para verificar el comportamiento del sistema.
3. **Resultados de pruebas**: Se incluirán los resultados de las pruebas realizadas, tanto manuales como automatizadas, para asegurar que todas las funcionalidades están operativas.

## Cómo usar las evidencias

- Las capturas de pantalla deben ser revisadas en el orden indicado durante la defensa del proyecto.
- Los logs deben ser analizados para identificar cualquier error o advertencia que haya surgido durante las pruebas.
- Los resultados de las pruebas deben ser discutidos para demostrar la efectividad de las implementaciones realizadas.
